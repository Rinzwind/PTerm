# PTerm

Using Unix terminal from Pharo.

Require libc compliant with the host pharo VM (32 or 64 bits) to be installed.

![PTerm](https://github.com/lxsang/PTerm/raw/master/Pterm.gif)

Install on pharo 7

```smalltalk
Metacello new
	repository: 'github://lxsang/PTerm';
	baseline:'PTerm';
	load
```
